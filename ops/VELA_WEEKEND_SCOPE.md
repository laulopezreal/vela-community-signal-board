# VELA Weekend Build Scope

## Challenge Context
Build an app that serves a community you care about.

## Concrete MVP (selected)
**Project:** Community Signal Board
**Audience:** Small operator communities (founders/builders) who need high-signal updates without noise.

## Problem
Community members miss important opportunities because updates are scattered across channels.

## MVP Solution
A lightweight web app that:
1. Ingests manual entries (title, source, urgency, category).
2. Ranks items by urgency and relevance.
3. Shows a clean dashboard with filters.
4. Exports daily digest markdown.

## Non-goals (for weekend)
- Multi-tenant auth complexity
- Advanced analytics
- Deep integrations requiring long setup

## Delivery Plan
- Day 1: scaffold app + core data model + dashboard
- Day 2: ranking + filters + digest export + final polish
- Day 3 buffer: submission assets and fixes only
